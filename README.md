# AI Chatbot iOS App

An AI-powered iOS mobile chat application built using Swift & SwiftUI.  
Designed for sales teams and customers to interact with AI through text, voice, and chat history.

## Features
- SwiftUI clean UI
- Multi-modal input (text + voice)
- Chat session history
- Smart suggestions
- Dynamic + static AI responses
- Smooth UI animations
- Tab-based navigation

## Requirements
- iOS 17+
- Xcode 15+
- Swift 5.9+

## Installation
1. Clone the repository:
   git clone https://github.com/shivani-sisodiya/AIChatBot-iOS.git
2. Open the project in Xcode
3. Run on Simulator or physical device
